        <div id="more" class="vditor-reset post__content"><h3>一、java8安装</h3>
<h5>1 Java 8 下载地址</h5>
<p>链接：<a href="https://pan.baidu.com/s/1NN4XBL5g1Xn7EwzM4YET0g">https://pan.baidu.com/s/1NN4XBL5g1Xn7EwzM4YET0g</a>
提取码：m4mq</p>
<h5>2 以root用户登录将下载的jdk-8u92-linux-x64.tar.gz文件放到~/data/soft/目录下，使用如下命令解压</h5>
<pre><code>
tar zxvf jdk-8u92-linux-x64.tar.gz -C ~/data/soft


</code></pre>
<h5>3 将java目录添加到etc/profile文件中</h5>
<pre><code>export JAVA_HOME=/root/data/jdk1.8
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH
</code></pre>
<h5>4 验证java</h5>
<pre><code>java -version
</code></pre>
<h3>二、redis安装</h3>
<h1>安装Redis服务器端</h1>
<pre><code>sudo apt-get install redis-server
</code></pre>
<p>安装完成后，Redis服务器会自动启动，我们检查Redis服务器程序</p>
<h2>检查Redis服务器系统进程</h2>
<pre><code>redis     9839  0.0  0.1  40136  3096 ?        Ssl  11:42   0:00 /usr/bin/redis-server 127.0.0.1:6379
root     10139  0.0  0.0  14228   912 pts/1    S+   11:42   0:00 grep --color=auto redis
</code></pre>
<h2>通过启动命令检查Redis服务器状态</h2>
<pre><code>netstat -nlt|grep 6379
tcp        0      0 127.0.0.1:6379          0.0.0.0:*               LISTEN
</code></pre>

<h1>通过命令行客户端访问Redis</h1>
<p>安装Redis服务器，会自动地一起安装Redis命令行客户端程序。
在本机输入redis-cli命令就可以启动，客户端程序访问Redis服务器。</p>
<pre><code>~ redis-cli
redis 127.0.0.1:6379&gt;

# 命令行的帮助

redis 127.0.0.1:6379&gt; help
redis-cli 2.2.12
Type: &quot;help @&quot; to get a list of commands in
      &quot;help &quot; for help on
      &quot;help &quot; to get a list of possible help topics
      &quot;quit&quot; to exit


# 查看所有的key列表

redis 127.0.0.1:6379&gt; keys *
(empty list or set)
</code></pre>
<h1>修改Redis的配置</h1>
<h2>使用Redis的访问账号</h2>
<p>默认情况下，访问Redis服务器是不需要密码的，为了增加安全性我们需要设置Redis服务器的访问密码。设置访问密码为redisredis。</p>
<p>用vi打开Redis服务器的配置文件redis.conf</p>
<pre><code>sudo vi /etc/redis/redis.conf

#取消注释requirepass
requirepass redisredis
</code></pre>
<h2>让Redis服务器被远程访问</h2>
<p>默认情况下，Redis服务器不允许远程访问，只允许本机访问，所以我们需要设置打开远程访问的功能。
用vi打开Redis服务器的配置文件redis.conf</p>
<pre><code>sudo vi /etc/redis/redis.conf

#注释bind
#bind 127.0.0.1
</code></pre>
<p>修改后，重启Redis服务器。</p>
<pre><code>sudo /etc/init.d/redis-server restart
Stopping redis-server: redis-server.
Starting redis-server: redis-server.
</code></pre>
<p>我们在远程的另一台Linux访问Redis服务器</p>
<pre><code>redis-cli -a redisredis -h 192.168.1.199

redis 192.168.1.199:6379&gt; keys *
1) &quot;key2&quot;
2) &quot;key3&quot;
</code></pre>
