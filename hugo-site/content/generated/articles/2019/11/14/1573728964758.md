        <div id="more" class="vditor-reset post__content"><h5>1.查看php版本</h5>
<pre><code>php -v
当前7.2版本
</code></pre>
<h5>2.查看是否安装mysql扩展</h5>
<p>两种方式</p>
<pre><code>php -m
php -r 'phpinfo();'
#查看加载顺序
grep -Hrv &quot;;&quot; /etc/php | grep -E &quot;extension(\s+)?=&quot;
</code></pre>
<h5>3. 安装mysql扩展</h5>
<pre><code>sudo apt install php7.2-mysql
</code></pre>
<h5>4. 修改配置文件</h5>
<pre><code> cd /etc/php/7.2/cli //进入配置文件目录
 sudo vim php.ini //vim打开配置文件
 //可能会输入root用户密码
 /pdo //查找，输入后按enter键即可
 //按i键进入vim编辑模式
 extension=php_pdo_mysql.dll //去掉extensions前面的;号
 //按shift + : 号，然后输入wq
</code></pre>
