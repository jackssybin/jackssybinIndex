<h3>一、安装Chrome浏览器</h3>
<p><strong>1、安装依赖</strong></p>
<blockquote>
<pre><code>sudo apt-get install libxss1 libappindicator1 libindicator7
</code></pre>
</blockquote>
<p><strong>2、下载Chrome安装包</strong></p>
<blockquote>
<pre><code>wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
</code></pre>
</blockquote>
<p><strong>3、安装</strong></p>
<blockquote>
<pre><code>sudo dpkg -i google-chrome*.deb

</code></pre>
<pre><code>sudo apt-get install -f
</code></pre>
</blockquote>
<p><strong>4、ubuntu如何安装lsb_release工具?</strong></p>
<pre><code>sudo apt-get install lsb-core -y
</code></pre>
<h3><strong>二、安装ChromeDriver</strong></h3>
<p><strong>1、安装<code>xvfb</code>以便我们可以无头奔跑地运行Chrome</strong></p>
<blockquote>
<pre><code>sudo apt-get install xvfb
</code></pre>
</blockquote>
<p><strong>2、安装依赖</strong></p>
<blockquote>
<pre><code>sudo apt-get install unzip
</code></pre>
</blockquote>
<p><strong>3、下载安装包</strong></p>
<blockquote>
<pre><code>wget -N http://chromedriver.storage.googleapis.com/2.26/chromedriver_linux64.zip

</code></pre>
</blockquote>
<p><strong><code>4、解压缩+添加执行权限</code></strong></p>
<blockquote>
<pre><code>unzip chromedriver_linux64.zip
</code></pre>
</blockquote>
<p><strong>5、移动</strong></p>
<blockquote>
<pre><code>sudo mv -f chromedriver /usr/local/share/chromedriver
</code></pre>
</blockquote>
<p><strong>6、建立软连接</strong></p>
<blockquote>
<pre><code>sudo ln -s /usr/local/share/chromedriver /usr/local/bin/chromedriver
sudo ln -s /usr/local/share/chromedriver /usr/bin/chromedriver
</code></pre>
</blockquote>
<h3>三、无头运行Chrome</h3>
<p><strong>1、安装Python依赖</strong></p>
<blockquote>
<p>pip3 install selenium</p>
<p>pip3 install pyvirtualdisplay</p>
</blockquote>
<p><strong>2、开干</strong></p>
<pre><code>from selenium import webdriver
driver = webdriver.Chrome() # 初始化Chrome
driver.get('https://jackssybin.cn/articles/2019/11/04/1572870368019.html')
print(driver.title)

</code></pre>
