      <div><strong>导读</strong><span>本文归入「工具、效率与博客建设」专题，主要记录：1.社区 https://rustlangcn.org/ 2.环境安装 windows 下载地址 https://www.rustlang.org/install.html 下载完之后》rustupinit.exe 直接安装走默认的就可以了 安装完之后看看 版本，确定下是否安装成功了 rustc version 我安装...</span></div>

        <div id="more" class="vditor-reset post__content"><p><img src="https://img.hacpai.com/bing/20190719.jpg?imageView2/1/w/960/h/540/interlace/1/q/100" alt=""></p>
<h2>1.社区</h2>
<p><a href="https://rustlang-cn.org/">https://rustlang-cn.org/</a></p>
<h2>2.环境安装</h2>
<p>windows 下载地址
<a href="https://www.rust-lang.org/install.html">https://www.rust-lang.org/install.html</a>
<img src="https://img.hacpai.com/file/2019/09/1-fe9c0ecb.png" alt="1.png"></p>
<p>下载完之后》rustup-init.exe
直接安装走默认的就可以了</p>
<p>安装完之后看看 版本，确定下是否安装成功了</p>
<pre><code>rustc --version
</code></pre>
<p>我安装完，报错了
no default toolchain configured
执行命令</p>
<p>更行版本</p>
<pre><code>rustup update
</code></pre>
<p>卸载rust</p>
<pre><code>rustup self uninstall
</code></pre>
<h3>3.hello word 走一波</h3>
<p>新建文件main.rs编写程序</p>
<pre><code>fn main() {
    println!(&quot;Hello, world!&quot;);
}

</code></pre>
<p>然后编译一下 rustc <a href="http://main.rs">main.rs</a>
生成编译文件 和可执行文件如图
<img src="https://img.hacpai.com/file/2019/09/image-dcafecf4.png" alt="image.png"></p>
<p>看生成了 main.exe 文件
windows用.\main.exe 执行一下这个可执行文件。看看效果哈。
<img src="https://img.hacpai.com/file/2019/09/image-13b8cd12.png" alt="image.png">
熟悉的世界你好。跃然屏幕上。心情很好。没遇到啥问题很好哈</p>
<p>学习了hello word 然后再看看rust的依赖包管理工具cargo</p>
<h3>4.hello cargo 接着干</h3>
<p>Cargo 是 Rust 的构建系统和包管理器，它可以处理很多任务，比如构建代码、下载依赖库并编译这些库。</p>
<p>官方默认是安装了的  --version查看一下</p>
<pre><code>cargo --version
</code></pre>
<h3>5.使用 Cargo 创建项目</h3>
<pre><code>cargo new hello_cargo
</code></pre>
<p>运行完这个命令会自动给你生成一个helloword文件的哈。切换到跟目录build一下</p>
<pre><code>cargo build
</code></pre>
<p>效果图如下
<img src="https://img.hacpai.com/file/2019/09/image-6e46b52e.png" alt="image.png">
cargo.lock文件系统用的，不用管他，也不要修改他
生成的文件在target\debug下面</p>
<pre><code>.\target\debug\hello_cargo.exe
如果只有一个运行文件的话
cargo run 也能达到相同目的
cargo check 是在运行之前做一下检查。校验。
</code></pre>
<p>熟悉的helloword 又出现了。</p>
<p>接着说下发布。</p>
<pre><code>cargo build  #这个是开发时用的，编译快
cargo build --release #这个时发布时用的，编译可能会慢，但是cargo里内部做优化了
</code></pre>
<h2>总结一下啊</h2>
<ol>
<li>rust安装</li>
<li>rust运行命令 rustc <a href="http://main.rs">main.rs</a></li>
<li>cargo命令：cargo --version, cargo new hello_cargo,cargo build,cargo check,cargo run,cargo build --release</li>
</ol>
