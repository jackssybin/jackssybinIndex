<h1>常用模块</h1>
<p>socket模块</p>
<p>常用于通讯，任何通讯工具中都含有socket，比如qq，微信。</p>
<p>udp实例：</p>
<h1>导入模块</h1>
<p>import socket</p>
<p>def main():</p>
<pre><code># 创建套接字

# 参数一：ip协议，socket.AF_INET表示ipv4协议。

# 参数二：使用udp协议还是tcp协议 socket.SOCK_DGRAM表示udp协议。

udp_socket = socket.socket(socket.AF_INET,socket.SOCK_DGRAM)

# 绑定信息，里面是一个元组，第一个参数为ip，&quot;&quot;表示自己的ip

# 第二个表示端口号

udp_socket.bind((&quot;&quot;,7890))

# 接收对方的ip地址

dest_ip = input(&quot;请输入对方的ip：&quot;)

# 接收对方的端口(port)

dest_port = int(input(&quot;请输入对方的接收端口:&quot;))

# 接收发送消息

send_data = input(&quot;请输入需要发送的消息：&quot;)

# 发送消息

# 参数一：表示需要发送的消息。

# 参数二：一个元组，第一个为对方的ip地址，第二个为对方的端口号

udp_socket.sendto(send_data.encode(&quot;utf-8&quot;),(dest_ip,dest_port))

# 接收消息 1024表示能接收的最大值。

recv_data = udp_socket.recvfrom(1024)

# 输出接收到的消息，返回的也是一个元组，1，发送过来的信息2，表示发送方的ip和端口

# 为什么需要解码gbk？因为Windows的编码为gbk

print(recv_data[0].decode(&quot;gbk&quot;))

# 关闭套接字

udp_socket.close()
</code></pre>

<p>可以看到上文需要输入对方的ip地址和端口号，为什么？</p>
<p>简单来说就是通过ip地址找到你的电脑，再通过端口号找到接收的程序。</p>
<p>其他的注释应该很清楚了。tcp和udp的区别？</p>
<p>tcp是传输比较稳定，不掉包，udp是传输快，容易掉包。</p>
<p>掉包的意思就是说发送过去，对方一不定可以接收到（当然数据过大的时候会出现这种情况。）</p>
<p>这里补充几个名词的含义。</p>
<p>单工：比如说收音机，只能发，或者只能收消息。</p>
<p>半双工：能发也能收，但是在同一时刻只能收或只能发。</p>
<p>全双工：同一时刻能发也能收。</p>
<p>tcp实例</p>
<p>import socket</p>
<p>def main():</p>
<pre><code># 创建套接字

# 参数一：ip协议，socket.AF_INET表示ipv4协议。

# 参数二：使用udp协议还是tcp协议 socket.SOCK_STREAM表示tcp协议

tcp_socket = socket.socket(socket.AF_INET,socket.SOCK_STREAM)

# 绑定信息，里面是一个元组，第一个参数为ip，&quot;&quot;表示自己的ip

# 第二个表示端口号

tcp_socket.bind((&quot;&quot;,7890))

# 接收对方的ip地址

dest_ip = input(&quot;请输入对方的ip：&quot;)

# 接收对方的端口(port)

dest_port = int(input(&quot;请输入对方的接收端口:&quot;))

# 链接服务器

tcp_socket.connec((dest_ip,dest_port))

# 接收发送消息

send_data = input(&quot;请输入需要发送的消息：&quot;)

# 发送消息

tcp_socket.send(send_data.encode(&quot;utf-8&quot;))

# 接收消息 1024表示能接收的最大值。

recv_data = tcp_socket.recv(1024)

# 输出接收到的消息，返回的也是一个元组，1，发送过来的信息2，表示发送方的ip和端口

# 为什么需要解码gbk？因为Windows的编码为gbk

print(recv_data[0].decode(&quot;gbk&quot;))

# 关闭套接字

tcp_socket.close()
</code></pre>

<p>time模块</p>
<p>import time #导入模块</p>
<p>time.sleep(5) # 停留5秒，不给钱不优化那种</p>
<p>time.time() # 返回从1970年1月1日到现在经历了多少秒。</p>
<h1>格式化时间</h1>
<p>time.strftime(&quot;%Y-%m-%d %H:%M:%S&quot;) # 2018-11-08 21:50:01</p>
<p>time.strftime(&quot;%Y/%m/%d %H:%M:%S&quot;) # 2018/11/08 21:50:01</p>
<p>其他格式：</p>
<p>%Y 四位数的年份表示（000-9999）</p>
<p>%m 月份（01-12）</p>
<p>%d 月内中的一天（0-31）</p>
<p>%H 24小时制小时数（0-23）</p>
<p>%I 12小时制小时数（01-12）</p>
<p>%M 分钟数（00=59）</p>
<p>%S 秒（00-59）</p>
<p>%a 本地简化星期名称</p>
<p>%A 本地完整星期名称</p>
<p>%b 本地简化的月份名称</p>
<p>%B 本地完整的月份名称</p>
<p>%c 本地相应的日期表示和时间表示</p>
<p>%j 年内的一天（001-366）</p>
<p>%p 本地A.M.或P.M.的等价符</p>
<p>%U 一年中的星期数（00-53）星期天为星期的开始</p>
<p>%w 星期（0-6），星期天为星期的开始</p>
<p>%W 一年中的星期数（00-53）星期一为星期的开始</p>
<p>%x 本地相应的日期表示</p>
<p>%X 本地相应的时间表示</p>
<p>%Z 当前时区的名称</p>
<p>%% %号本身</p>

<h1>当我们想要像调用列表一样调用上面的字符串时，我们需要反序列化</h1>
<h1>同理 当我们将比如列表转换为字符串类型就称之为序列化</h1>

<p>json:</p>
<h1>导入模块</h1>
<pre><code>import json # 可以转化的有 数字，字符串，列表，字典，元组（转化为列表）

# 序列化

list = [1,2,3,4,5]

print(type(list)) # &lt;class 'list'&gt; 列表类型

str = json.dumps(list)

print(type(str)) # &lt;class 'str'&gt; 此时就转化为了字符串格式

# 反序列化

attr = json.loads(str)

print(type(attr)) # &lt;class 'list'&gt; 列表类型

# 序列化写入文件

f = open('txt','w',encoding = 'utf-8')

json.dump(list,f，ensure_ascii=False) # 把list 序列化写入文件 ensure_ascii=False如果有中文需要加上

f.close()

# 从文件里拿出数据反序列化

f = open('txt','r',encoding = 'utf-8')

txt =json.load(f)

f.close()

print(type(txt)) # list 返回的list
</code></pre>
