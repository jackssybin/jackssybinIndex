      <div><strong>专题</strong><a href="/topics/python-crawler.html">Python 爬虫与自动化</a></div>
      <div><strong>导读</strong><span>本文归入「Python 爬虫与自动化」专题，主要记录：1.安装pyinstaller pip3 install pyinstaller pyinstaller onefile hello.py 2.生成exe 跳到python文件目录下面运行命令 pyinstaller onefile python文件名 onefile的作用是产生的结果汇成一个exe的文件,文件存放再d...</span></div>
  </section>
        
        <div id="more" class="vditor-reset post__content"><p><strong>1.安装pyinstaller</strong></p>
<pre><code>pip3 install pyinstaller
</code></pre>
<p>pyinstaller --onefile <a href="http://hello.py">hello.py</a></p>
<p><strong>2.生成exe</strong>
跳到python文件目录下面运行命令 pyinstaller --onefile python文件名</p>
<p>--onefile的作用是产生的结果汇成一个exe的文件,文件存放再dist目录底下。</p>
<pre><code>pyinstaller --onefile hello.py
</code></pre>
<p><strong>3.执行exe</strong>
双击即可执行</p>
