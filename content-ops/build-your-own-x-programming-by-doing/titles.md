# 候选标题

## WeChat 候选
1. [对比落差] 学了三年编程我才明白：为什么 GitHub 上 53 万人 star 这个项目
2. [数字反直觉] 刷完 100 道 LeetCode 不如动手造这 30 个项目
3. [踩坑复盘] 看完 20 本编程书还是不会写项目，朋友甩给我这个仓库
4. [身份对号入座] 如果你觉得编程"一看就会，一写就废"，一定要看看这个仓库
5. [反问反常识] 费曼说"不能创造就不能理解"，这个仓库把这句话变成了 500+ 教程

## Zhihu 候选（更冷静、结论前置）
1. 如何评价 GitHub 上 53 万星的 build-your-own-x？真的能学会编程吗？
2. build-your-own-x 值得刷吗？我的结论是：分人群看

4. 学编程卡在"看得懂，不会写"这个瓶颈，如何突破？

## 最终选定
- WeChat: **看完 20 本编程书还是不会写项目，朋友甩给我这个仓库** — 理由：直接戳中绝大多数学习者的痛点，"看完书不会写项目"是非常普遍的焦虑，这个钩子能勾起点击欲望
- Zhihu: **如何评价 GitHub 上 53 万星的 build-your-own-x？真的能学会编程吗？** — 理由：知乎用户喜欢看评价类问题，开门见山给结论，符合知乎流量逻辑
