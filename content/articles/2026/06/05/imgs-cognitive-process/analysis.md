# analysis

- Content type: 知乎回答 / 方法论文章
- Purpose: 回答“怎样提高自己的认知流程”，避免空泛鸡汤，给出可执行流程。
- Background references:
  1. 达尔文：重视记录反证，避免遗忘不支持自己理论的事实。
  2. 富兰克林：用赞成/反对两列清单辅助重大判断。
  3. 查理·芒格：强调避免愚蠢和常见误判，而不是追求神秘顿悟。
- Core argument: 认知不是玄学，而是“写清问题 → 区分事实和观点 → 主动找反证 → 小实验 → 复盘判断过程”的工作流。
- Illustration settings: mixed, balanced density, editorial vector, warm/blue/green palette.
