<h2>29.1 用户管理 ⭐⭐⭐⭐⭐</h2>
<h3>29.1.1 创建用户</h3>
<p><strong>基本语法：</strong></p>
<pre><code class="language-sql">CREATE USER 'username'@'host' IDENTIFIED BY 'password';
<p>-- 示例：
-- 创建本地用户
CREATE USER 'app_user'@'localhost' IDENTIFIED BY 'App@123456';</p>
<p>-- 创建远程用户
CREATE USER 'app_user'@'192.168.1.100' IDENTIFIED BY 'App@123456';</p>
<p>-- 创建任意主机可访问的用户
CREATE USER 'app_user'@'%' IDENTIFIED BY 'App@123456';</p>
<p>-- 创建指定网段可访问的用户
CREATE USER 'app_user'@'192.168.1.%' IDENTIFIED BY 'App@123456';
</code></pre></p>
<h3>29.1.2 查看用户</h3>
<p><strong>查看所有用户：</strong></p>
<pre><code class="language-sql">-- 查看用户列表
SELECT user, host FROM mysql.user;
<p>-- 查看当前用户
SELECT USER(), CURRENT_USER();</p>
<p>-- 查看用户详细信息
SELECT * FROM mysql.user WHERE user = 'app_user'\G
</code></pre></p>
<h3>29.1.3 修改用户</h3>
<p><strong>修改密码：</strong></p>
<pre><code class="language-sql">-- 修改自己的密码
ALTER USER USER() IDENTIFIED BY 'NewPassword@123';
<p>-- 修改其他用户密码（需要权限）
ALTER USER 'app_user'@'localhost' IDENTIFIED BY 'NewPassword@123';</p>
<p>-- 使用SET PASSWORD（旧方法）
SET PASSWORD FOR 'app_user'@'localhost' = PASSWORD('NewPassword@123');
</code></pre></p>
<p><strong>修改用户名：</strong></p>
<pre><code class="language-sql">RENAME USER 'old_user'@'localhost' TO 'new_user'@'localhost';
</code></pre>
<h3>29.1.4 删除用户</h3>
<p><strong>删除用户：</strong></p>
<pre><code class="language-sql">DROP USER 'app_user'@'localhost';
<p>-- 删除多个用户
DROP USER 'user1'@'localhost', 'user2'@'localhost';</p>
<p>-- 如果用户不存在不报错
DROP USER IF EXISTS 'app_user'@'localhost';
</code></pre></p>

<h2>29.2 权限管理 ⭐⭐⭐⭐⭐</h2>
<h3>29.2.1 权限类型</h3>
<p><strong>常用权限：</strong></p>
<ul>
<li><code>ALL PRIVILEGES</code>：所有权限</li>
<li><code>SELECT</code>：查询权限</li>
<li><code>INSERT</code>：插入权限</li>
<li><code>UPDATE</code>：更新权限</li>
<li><code>DELETE</code>：删除权限</li>
<li><code>CREATE</code>：创建数据库/表权限</li>
<li><code>DROP</code>：删除数据库/表权限</li>
<li><code>ALTER</code>：修改表结构权限</li>
<li><code>INDEX</code>：创建/删除索引权限</li>
<li><code>EXECUTE</code>：执行存储过程/函数权限</li>
<li><code>GRANT OPTION</code>：授权权限</li>
</ul>
<h3>29.2.2 授予权限</h3>
<p><strong>全局权限：</strong></p>
<pre><code class="language-sql">-- 授予所有权限
GRANT ALL PRIVILEGES ON *.* TO 'admin'@'localhost';
<p>-- 授予特定权限
GRANT SELECT, INSERT, UPDATE ON <em>.</em> TO 'app_user'@'localhost';
</code></pre></p>
<p><strong>数据库权限：</strong></p>
<pre><code class="language-sql">-- 授予数据库所有权限
GRANT ALL PRIVILEGES ON mydb.* TO 'app_user'@'localhost';
<p>-- 授予数据库特定权限
GRANT SELECT, INSERT, UPDATE, DELETE ON mydb.* TO 'app_user'@'localhost';
</code></pre></p>
<p><strong>表权限：</strong></p>
<pre><code class="language-sql">-- 授予表所有权限
GRANT ALL PRIVILEGES ON mydb.users TO 'app_user'@'localhost';
<p>-- 授予表特定权限
GRANT SELECT, INSERT ON mydb.users TO 'app_user'@'localhost';
</code></pre></p>
<p><strong>列权限：</strong></p>
<pre><code class="language-sql">-- 授予列权限
GRANT SELECT (id, name), UPDATE (name) ON mydb.users TO 'app_user'@'localhost';
</code></pre>
<p><strong>存储过程权限：</strong></p>
<pre><code class="language-sql">-- 授予执行存储过程权限
GRANT EXECUTE ON PROCEDURE mydb.sp_get_user TO 'app_user'@'localhost';
</code></pre>
<p><strong>授权并允许转授：</strong></p>
<pre><code class="language-sql">-- WITH GRANT OPTION：允许用户将权限授予其他用户
GRANT SELECT, INSERT ON mydb.* TO 'app_user'@'localhost' WITH GRANT OPTION;
</code></pre>
<h3>29.2.3 查看权限</h3>
<p><strong>查看用户权限：</strong></p>
<pre><code class="language-sql">-- 查看当前用户权限
SHOW GRANTS;
<p>-- 查看指定用户权限
SHOW GRANTS FOR 'app_user'@'localhost';</p>
<p>-- 查看权限详细信息
SELECT * FROM mysql.user WHERE user = 'app_user'\G
SELECT * FROM mysql.db WHERE user = 'app_user'\G
SELECT * FROM mysql.tables_priv WHERE user = 'app_user'\G
</code></pre></p>
<h3>29.2.4 撤销权限</h3>
<p><strong>撤销权限：</strong></p>
<pre><code class="language-sql">-- 撤销所有权限
REVOKE ALL PRIVILEGES ON *.* FROM 'app_user'@'localhost';
<p>-- 撤销特定权限
REVOKE INSERT, UPDATE ON mydb.* FROM 'app_user'@'localhost';</p>
<p>-- 撤销表权限
REVOKE SELECT ON mydb.users FROM 'app_user'@'localhost';
</code></pre></p>
<h3>29.2.5 刷新权限</h3>
<p><strong>刷新权限：</strong></p>
<pre><code class="language-sql">-- 刷新权限（使权限立即生效）
FLUSH PRIVILEGES;
<p>-- 注意：使用GRANT/REVOKE会自动刷新，直接修改mysql.user表需要手动刷新
</code></pre></p>

<h2>29.3 角色管理（MySQL 8.0+）⭐⭐⭐⭐</h2>
<h3>29.3.1 创建角色</h3>
<p><strong>创建角色：</strong></p>
<pre><code class="language-sql">-- 创建角色
CREATE ROLE 'app_read', 'app_write', 'app_admin';
<p>-- 授予角色权限
GRANT SELECT ON mydb.* TO 'app_read';
GRANT INSERT, UPDATE, DELETE ON mydb.* TO 'app_write';
GRANT ALL PRIVILEGES ON mydb.* TO 'app_admin';
</code></pre></p>
<h3>29.3.2 分配角色</h3>
<p><strong>分配角色给用户：</strong></p>
<pre><code class="language-sql">-- 创建用户
CREATE USER 'user1'@'localhost' IDENTIFIED BY 'Password@123';
<p>-- 分配角色
GRANT 'app_read' TO 'user1'@'localhost';</p>
<p>-- 分配多个角色
GRANT 'app_read', 'app_write' TO 'user1'@'localhost';</p>
<p>-- 激活角色（重要）
SET DEFAULT ROLE ALL TO 'user1'@'localhost';
</code></pre></p>
<h3>29.3.3 查看角色</h3>
<p><strong>查看角色：</strong></p>
<pre><code class="language-sql">-- 查看所有角色
SELECT user, host FROM mysql.user WHERE account_locked = 'Y';
<p>-- 查看用户的角色
SHOW GRANTS FOR 'user1'@'localhost';</p>
<p>-- 查看角色的权限
SHOW GRANTS FOR 'app_read';
</code></pre></p>
<h3>29.3.4 删除角色</h3>
<p><strong>删除角色：</strong></p>
<pre><code class="language-sql">-- 撤销用户的角色
REVOKE 'app_read' FROM 'user1'@'localhost';
<p>-- 删除角色
DROP ROLE 'app_read';
</code></pre></p>

<h2>29.4 权限最佳实践</h2>
<h3>29.4.1 最小权限原则</h3>
<p><strong>原则：</strong></p>
<ul>
<li>只授予必要的权限</li>
<li>不要使用root用户</li>
<li>不要授予ALL PRIVILEGES</li>
</ul>
<p><strong>示例：</strong></p>
<pre><code class="language-sql">-- ❌ 不推荐：授予所有权限
GRANT ALL PRIVILEGES ON *.* TO 'app_user'@'%';
<p>-- ✅ 推荐：只授予必要权限
GRANT SELECT, INSERT, UPDATE, DELETE ON mydb.* TO 'app_user'@'192.168.1.%';
</code></pre></p>
<h3>29.4.2 用户分类</h3>
<p><strong>只读用户：</strong></p>
<pre><code class="language-sql">CREATE USER 'readonly'@'%' IDENTIFIED BY 'ReadOnly@123';
GRANT SELECT ON mydb.* TO 'readonly'@'%';
</code></pre>
<p><strong>读写用户：</strong></p>
<pre><code class="language-sql">CREATE USER 'readwrite'@'192.168.1.%' IDENTIFIED BY 'ReadWrite@123';
GRANT SELECT, INSERT, UPDATE, DELETE ON mydb.* TO 'readwrite'@'192.168.1.%';
</code></pre>
<p><strong>管理员用户：</strong></p>
<pre><code class="language-sql">CREATE USER 'admin'@'localhost' IDENTIFIED BY 'Admin@123';
GRANT ALL PRIVILEGES ON mydb.* TO 'admin'@'localhost';
</code></pre>
<p><strong>备份用户：</strong></p>
<pre><code class="language-sql">CREATE USER 'backup'@'localhost' IDENTIFIED BY 'Backup@123';
GRANT SELECT, LOCK TABLES, SHOW VIEW, EVENT, TRIGGER ON *.* TO 'backup'@'localhost';
</code></pre>
<p><strong>监控用户：</strong></p>
<pre><code class="language-sql">CREATE USER 'monitor'@'localhost' IDENTIFIED BY 'Monitor@123';
GRANT PROCESS, REPLICATION CLIENT ON *.* TO 'monitor'@'localhost';
</code></pre>
<h3>29.4.3 密码策略</h3>
<p><strong>设置密码策略：</strong></p>
<pre><code class="language-sql">-- 查看密码策略
SHOW VARIABLES LIKE 'validate_password%';
<p>-- 设置密码策略（my.cnf）
[mysqld]
validate_password.policy = MEDIUM
validate_password.length = 8
validate_password.mixed_case_count = 1
validate_password.number_count = 1
validate_password.special_char_count = 1</p>
<p>-- 密码过期策略
ALTER USER 'app_user'@'localhost' PASSWORD EXPIRE INTERVAL 90 DAY;</p>
<p>-- 密码重用策略
ALTER USER 'app_user'@'localhost' PASSWORD HISTORY 5;
</code></pre></p>

<h2>29.5 常见场景</h2>
<h3>29.5.1 应用程序用户</h3>
<p><strong>创建应用程序用户：</strong></p>
<pre><code class="language-sql">-- 1. 创建用户
CREATE USER 'app_user'@'192.168.1.%' IDENTIFIED BY 'App@123456';
<p>-- 2. 授予权限
GRANT SELECT, INSERT, UPDATE, DELETE ON mydb.* TO 'app_user'@'192.168.1.%';</p>
<p>-- 3. 刷新权限
FLUSH PRIVILEGES;
</code></pre></p>
<h3>29.5.2 主从复制用户</h3>
<p><strong>创建复制用户：</strong></p>
<pre><code class="language-sql">-- 主库创建复制用户
CREATE USER 'repl'@'192.168.1.%' IDENTIFIED BY 'Repl@123456';
GRANT REPLICATION SLAVE ON *.* TO 'repl'@'192.168.1.%';
FLUSH PRIVILEGES;
</code></pre>
