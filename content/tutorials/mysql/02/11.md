<h1>第11章：视图、存储过程与函数</h1>
<blockquote>
<p>提高代码复用性和安全性的高级特性</p>
</blockquote>
<h2>11.1 视图（View）⭐⭐⭐⭐</h2>
<h3>11.1.1 什么是视图？</h3>
<p><strong>定义：</strong></p>
<ul>
<li>虚拟表，基于SQL查询结果</li>
<li>不存储数据，只存储查询语句</li>
<li>简化复杂查询，提高安全性</li>
</ul>
<p><strong>示例：</strong></p>
<pre><code class="language-sql">-- 创建视图
CREATE VIEW v_user_orders AS
SELECT 
    u.id AS user_id,
    u.name AS user_name,
    o.id AS order_id,
    o.amount,
    o.create_time
FROM users u
LEFT JOIN orders o ON u.id = o.user_id;
<p>-- 使用视图
SELECT * FROM v_user_orders WHERE user_id = 1;
</code></pre></p>
<h3>11.1.2 视图操作</h3>
<p><strong>创建视图：</strong></p>
<pre><code class="language-sql">-- 基本语法
CREATE VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;
<p>-- 创建或替换视图
CREATE OR REPLACE VIEW v_active_users AS
SELECT id, name, email
FROM users
WHERE status = 'active';</p>
<p>-- 创建视图并指定列名
CREATE VIEW v_user_info(user_id, user_name, user_email) AS
SELECT id, name, email FROM users;
</code></pre></p>
<p><strong>查看视图：</strong></p>
<pre><code class="language-sql">-- 查看所有视图
SHOW FULL TABLES WHERE Table_type = 'VIEW';
<p>-- 查看视图定义
SHOW CREATE VIEW v_user_orders;</p>
<p>-- 查看视图结构
DESC v_user_orders;
</code></pre></p>
<p><strong>修改视图：</strong></p>
<pre><code class="language-sql">-- 方法1：CREATE OR REPLACE
CREATE OR REPLACE VIEW v_active_users AS
SELECT id, name, email, phone
FROM users
WHERE status = 'active';
<p>-- 方法2：ALTER VIEW
ALTER VIEW v_active_users AS
SELECT id, name, email, phone, address
FROM users
WHERE status = 'active';
</code></pre></p>
<p><strong>删除视图：</strong></p>
<pre><code class="language-sql">DROP VIEW IF EXISTS v_user_orders;
</code></pre>
<h3>11.1.3 可更新视图</h3>
<p><strong>可更新视图条件：</strong></p>
<ul>
<li>不包含聚合函数</li>
<li>不包含DISTINCT</li>
<li>不包含GROUP BY</li>
<li>不包含UNION</li>
</ul>
<p><strong>示例：</strong></p>
<pre><code class="language-sql">-- 可更新视图
CREATE VIEW v_users AS
SELECT id, name, email FROM users;
<p>-- 可以INSERT、UPDATE、DELETE
INSERT INTO v_users(name, email) VALUES ('张三', 'test@example.com');
UPDATE v_users SET name = '李四' WHERE id = 1;
DELETE FROM v_users WHERE id = 1;</p>
<p>-- 不可更新视图（包含聚合函数）
CREATE VIEW v_user_count AS
SELECT COUNT(*) AS total FROM users;</p>
<p>-- 无法INSERT、UPDATE、DELETE
</code></pre></p>
<h3>11.1.4 视图的优缺点</h3>
<p><strong>优点：</strong></p>
<ul>
<li>✅ 简化复杂查询</li>
<li>✅ 提高安全性（隐藏敏感字段）</li>
<li>✅ 逻辑独立性（表结构变化，视图不变）</li>
</ul>
<p><strong>缺点：</strong></p>
<ul>
<li>❌ 性能可能较差</li>
<li>❌ 不能创建索引</li>
</ul>

<h2>11.2 存储过程（Stored Procedure）⭐⭐⭐⭐</h2>
<h3>11.2.1 什么是存储过程？</h3>
<p><strong>定义：</strong></p>
<ul>
<li>一组预编译的SQL语句</li>
<li>存储在数据库中</li>
<li>可重复调用</li>
</ul>
<p><strong>优点：</strong></p>
<ul>
<li>✅ 减少网络传输</li>
<li>✅ 提高性能（预编译）</li>
<li>✅ 代码复用</li>
<li>✅ 安全性（权限控制）</li>
</ul>
<h3>11.2.2 创建存储过程</h3>
<p><strong>基本语法：</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE procedure_name(
[IN|OUT|INOUT] parameter_name datatype,
...
)
BEGIN
-- SQL语句
END$$</p>
<p>DELIMITER ;
</code></pre></p>
<p><strong>示例1：无参数存储过程</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE sp_get_user_count()
BEGIN
SELECT COUNT(*) AS total FROM users;
END$$</p>
<p>DELIMITER ;</p>
<p>-- 调用
CALL sp_get_user_count();
</code></pre></p>
<p><strong>示例2：IN参数（输入参数）</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE sp_get_user_by_id(
IN p_user_id INT
)
BEGIN
SELECT * FROM users WHERE id = p_user_id;
END$$</p>
<p>DELIMITER ;</p>
<p>-- 调用
CALL sp_get_user_by_id(1);
</code></pre></p>
<p><strong>示例3：OUT参数（输出参数）</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE sp_get_user_count_out(
OUT p_count INT
)
BEGIN
SELECT COUNT(*) INTO p_count FROM users;
END$$</p>
<p>DELIMITER ;</p>
<p>-- 调用
CALL sp_get_user_count_out(@count);
SELECT @count;
</code></pre></p>
<p><strong>示例4：INOUT参数（输入输出参数）</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE sp_double_value(
INOUT p_value INT
)
BEGIN
SET p_value = p_value * 2;
END$$</p>
<p>DELIMITER ;</p>
<p>-- 调用
SET @value = 10;
CALL sp_double_value(@value);
SELECT @value;  -- 20
</code></pre></p>
<h3>11.2.3 存储过程中的流程控制</h3>
<p><strong>IF语句：</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE sp_check_age(
IN p_age INT,
OUT p_result VARCHAR(20)
)
BEGIN
IF p_age &lt; 18 THEN
SET p_result = '未成年';
ELSEIF p_age &lt; 60 THEN
SET p_result = '成年';
ELSE
SET p_result = '老年';
END IF;
END$$</p>
<p>DELIMITER ;
</code></pre></p>
<p><strong>CASE语句：</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE sp_get_grade(
IN p_score INT,
OUT p_grade CHAR(1)
)
BEGIN
CASE
WHEN p_score &gt;= 90 THEN SET p_grade = 'A';
WHEN p_score &gt;= 80 THEN SET p_grade = 'B';
WHEN p_score &gt;= 70 THEN SET p_grade = 'C';
WHEN p_score &gt;= 60 THEN SET p_grade = 'D';
ELSE SET p_grade = 'F';
END CASE;
END$$</p>
<p>DELIMITER ;
</code></pre></p>
<p><strong>WHILE循环：</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE sp_sum_1_to_n(
IN p_n INT,
OUT p_sum INT
)
BEGIN
DECLARE i INT DEFAULT 1;
SET p_sum = 0;</p>
<pre><code>WHILE i &amp;lt;= p_n DO
    SET p_sum = p_sum + i;
    SET i = i + 1;
END WHILE;
</code></pre>
<p>END$$</p>
<p>DELIMITER ;
</code></pre></p>
<p><strong>REPEAT循环：</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE sp_sum_repeat(
IN p_n INT,
OUT p_sum INT
)
BEGIN
DECLARE i INT DEFAULT 1;
SET p_sum = 0;</p>
<pre><code>REPEAT
    SET p_sum = p_sum + i;
    SET i = i + 1;
UNTIL i &amp;gt; p_n
END REPEAT;
</code></pre>
<p>END$$</p>
<p>DELIMITER ;
</code></pre></p>
<p><strong>LOOP循环：</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE PROCEDURE sp_sum_loop(
IN p_n INT,
OUT p_sum INT
)
BEGIN
DECLARE i INT DEFAULT 1;
SET p_sum = 0;</p>
<pre><code>loop_label: LOOP
    SET p_sum = p_sum + i;
    SET i = i + 1;
<pre><code>IF i &amp;amp;gt; p_n THEN
    LEAVE loop_label;
END IF;
</code></pre>
<p>END LOOP;
</code></pre></p>
<p>END$$</p>
<p>DELIMITER ;
</code></pre></p>
<h3>11.2.4 存储过程管理</h3>
<p><strong>查看存储过程：</strong></p>
<pre><code class="language-sql">-- 查看所有存储过程
SHOW PROCEDURE STATUS WHERE Db = 'mydb';
<p>-- 查看存储过程定义
SHOW CREATE PROCEDURE sp_get_user_by_id;
</code></pre></p>
<p><strong>删除存储过程：</strong></p>
<pre><code class="language-sql">DROP PROCEDURE IF EXISTS sp_get_user_by_id;
</code></pre>

<h2>11.3 函数（Function）⭐⭐⭐⭐</h2>
<h3>11.3.1 存储函数 vs 存储过程</h3>
<table>
<thead>
<tr>
<th>特性</th>
<th>存储函数</th>
<th>存储过程</th>
</tr>
</thead>
<tbody>
<tr>
<td>返回值</td>
<td>必须返回一个值</td>
<td>可以返回多个值（OUT参数）</td>
</tr>
<tr>
<td>调用方式</td>
<td>SELECT中调用</td>
<td>CALL调用</td>
</tr>
<tr>
<td>参数</td>
<td>只有IN参数</td>
<td>IN、OUT、INOUT参数</td>
</tr>
<tr>
<td>事务</td>
<td>不能包含事务</td>
<td>可以包含事务</td>
</tr>
</tbody>
</table>
<h3>11.3.2 创建函数</h3>
<p><strong>基本语法：</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE FUNCTION function_name(
parameter_name datatype,
...
)
RETURNS return_datatype
[DETERMINISTIC | NOT DETERMINISTIC]
BEGIN
-- SQL语句
RETURN value;
END$$</p>
<p>DELIMITER ;
</code></pre></p>
<p><strong>示例：</strong></p>
<pre><code class="language-sql">DELIMITER $$
<p>CREATE FUNCTION fn_get_user_name(p_user_id INT)
RETURNS VARCHAR(100)
DETERMINISTIC
BEGIN
DECLARE v_name VARCHAR(100);
SELECT name INTO v_name FROM users WHERE id = p_user_id;
RETURN v_name;
END$$</p>
<p>DELIMITER ;</p>
<p>-- 调用
SELECT fn_get_user_name(1);
SELECT id, fn_get_user_name(id) AS name FROM orders;
</code></pre></p>
<h3>11.3.3 函数管理</h3>
<p><strong>查看函数：</strong></p>
<pre><code class="language-sql">-- 查看所有函数
SHOW FUNCTION STATUS WHERE Db = 'mydb';
<p>-- 查看函数定义
SHOW CREATE FUNCTION fn_get_user_name;
</code></pre></p>
<p><strong>删除函数：</strong></p>
<pre><code class="language-sql">DROP FUNCTION IF EXISTS fn_get_user_name;
</code></pre>
