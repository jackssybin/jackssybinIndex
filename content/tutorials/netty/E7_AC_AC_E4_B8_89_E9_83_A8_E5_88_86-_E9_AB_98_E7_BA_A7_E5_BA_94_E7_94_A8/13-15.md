<h2>第13章：流量整形与限流</h2>
<h3>13.1 为什么需要流量控制</h3>
<p><strong>问题</strong>：</p>
<ul>
<li>突发流量导致服务器过载</li>
<li>恶意攻击消耗资源</li>
<li>需要保护后端服务</li>
</ul>
<p><strong>解决方案</strong>：</p>
<ul>
<li>流量整形（Traffic Shaping）</li>
<li>限流（Rate Limiting）</li>
</ul>

<h3>14.3 防止 DDoS 攻击</h3>
<pre><code class="language-java">public class DDoSProtectionHandler extends ChannelInboundHandlerAdapter {
<pre><code>private static final Map&amp;lt;String, AtomicInteger&amp;gt; connectionCount = new ConcurrentHashMap&amp;lt;&amp;gt;();
private static final int MAX_CONNECTIONS_PER_IP = 10;
<p>@Override
public void channelActive(ChannelHandlerContext ctx) {
InetSocketAddress address = (InetSocketAddress) ctx.channel().remoteAddress();
String ip = address.getAddress().getHostAddress();</p>
<pre><code>AtomicInteger count = connectionCount.computeIfAbsent(ip, k -&amp;amp;gt; new AtomicInteger(0));

if (count.incrementAndGet() &amp;amp;gt; MAX_CONNECTIONS_PER_IP) {
    System.out.println(&amp;amp;quot;IP连接数超过限制，拒绝连接: &amp;amp;quot; + ip);
    ctx.close();
    count.decrementAndGet();
    return;
}

// 连接关闭时减少计数
ctx.channel().closeFuture().addListener(future -&amp;amp;gt; {
    count.decrementAndGet();
});

ctx.fireChannelActive();
</code></pre>
<p>}
